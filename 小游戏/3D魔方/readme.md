# Rubik Cube

ThreeJs 制作的魔方游戏，支持自定义魔方阶级（目前界面上只开放 2 - 10 阶魔方）。

## Demo

试玩地址：[Rubiks Cube - 魔方](https://pengfeiw.github.io/minicode/threejs-rubik)

## 运行项目

1. 安装

```shell
npm install
```

2. 运行

```shell
npm run start
```

3. 打开浏览器

浏览器输入 `http://localhost:8000`。
